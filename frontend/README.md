## Интеграция с API

Приложение интегрируется с backend на FastAPI со следующими эндпоинтами:

### Аутентификация

* `POST /api/auth/register` - регистрация нового пользователя
* `POST /api/auth/login` - вход с логином и паролем
* `POST /api/auth/refresh` - обновление access-токена
* `GET /api/auth/users/me` - получение данных текущего пользователя

### Преподаватели

* `GET /api/teachers` - список всех преподавателей
* `GET /api/teachers/{teacher_id}/availability` - получение доступных слотов преподавателя
* `POST /api/teachers/{teacher_id}/availability` - создание слота доступности (только для преподавателя)
* `PUT /api/teachers/availability/{id}` - редактирование слота (только для преподавателя)
* `DELETE /api/teachers/availability/{id}` - удаление слота (только для преподавателя)

### Бронирования

* `POST /api/bookings` - создание брони (только для студентов)
* `GET /api/bookings` - получение бронирований пользователя
* `PUT /api/bookings/{id}/confirm` - подтверждение брони (только для преподавателя)
* `DELETE /api/bookings/{id}` - отмена брони

### Студенты

* `GET /api/students/my-bookings` - получение бронирований студента

## Основные функции

### Автообновление токенов

Приложение автоматически обновляет истёкшие access-токены с помощью refresh-токена. Реализовано в `src/lib/api.ts` через Axios интерсепторы.

### Защищённые маршруты

Маршруты защищены на основе статуса аутентификации и роли пользователя с использованием компонента `ProtectedRoute`.

### Управление состоянием

Глобальное состояние аутентификации управляется через Zustand:

* Информация о пользователе
* Статус аутентификации
* Функции входа и выхода
* Автоматическое восстановление сессии

### Адаптивный дизайн

Все страницы полностью адаптивны:

* Мобильная-first верстка
* Брейкпоинты для планшетов и десктопов
* Удобный интерфейс для тач-устройств
* Скрываемая мобильная навигация
